GitHub-koppeling

Nodig om templates te lezen en op te slaan vanuit jullie repository. De token wordt enkel in jouw eigen browser bewaard (localStorage) — nooit verstuurd naar iemand anders dan GitHub zelf.

Token aanmaken: GitHub → je profielfoto → Settings → Developer settings → Fine-grained tokens → Generate new token. Beperk "Repository access" tot enkel deze repository, en geef bij "Permissions" → "Contents" de rechten Read and write.
Enkel lezen/kopiëren nodig (niet bewerken)? Dan volstaat een token zonder schrijfrechten, of laat dit veld leeg als de repository publiek is.
